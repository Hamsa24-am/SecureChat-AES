# 🔐 SecureChat-AES

> A futuristic Android encrypted messaging prototype built using Java, XML, and AES encryption.

---

## 🚀 Overview

SecureChat-AES is a cybersecurity-themed Android chat application that demonstrates secure message transmission using AES encryption and decryption.

The application simulates a secure communication environment with a modern cyberpunk-inspired UI, encrypted packet visualization, decrypted message rendering, and intelligent secure-channel responses.

Designed as an academic cybersecurity project, the app focuses on:

* secure messaging concepts
* AES encryption workflow
* Android UI/UX design
* encrypted communication simulation

---

## ✨ Features

* 🔒 AES-based message encryption & decryption
* 💬 Real-time encrypted message display
* 📡 Simulated secure incoming/outgoing transmissions

## ⚠ Security Note

This project is an educational prototype.

The AES key is currently hardcoded for demonstration purposes.

Real-world secure messaging applications should use:

* Android Keystore
* Secure key exchange
* Dynamic IV generation
* End-to-end encrypted sockets
* Authentication systems
